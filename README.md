# chatty
A simple split-pane chat interface for mocking a conversation between users.

## Installation

```sh
git clone git@github.com:orrybaram/chatty
cd chatty
npm i
# Grab some coffee
npm start
open http://localhost:3000
```
Open up localhost:3000 and take a look!
